Bromine

Let's create documentation

Bromine is the free tool for C/C++ programmers, to help to do documentation.

How to use Bromine? Just select all headers files (*.h / *.hpp) to documentate, where are documentation comments And press submit button!

How to create Bromine comments? Create default C/C++ comments and type the type of object in brackets.

Table of all comments detectable by documentator

Start of new method [start]
Name of class where method is [cla][MathClass]
Description [des][Calculating Pi number]
Arguments [arg][a - First argument ; b - Second argument]
Returns [ret][Pi number as delta]
